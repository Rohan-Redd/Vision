Mission statement

“Empowering the visually impaired by being their virtual eyes”




Object Detection

Using one of the most prominent object detection datasets, Microsoft's Common Objects in Context dataset (COCO). VISION's object detection feature provides voice, visual, and text outputs to help the user understand what objects are in front of him/her.




Face Recognition

VISION's face recognition uses OpenCV's 'face recognition' model, which has a 99.38 percent accuracy, and provides voice, image, and text outputs to help the user determine if the person in front of them is familiar to them and, if so, their name.
